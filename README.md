## Quick Start

**1. Remix into Framer** *(fastest — your purchase receipt includes the Remix link)*

Click the Remix link from your receipt. Framer opens with a live copy already in your workspace. Skip to step 3.

**2. Or paste the component manually**

1. Open your Framer project → **Assets → Code → +** → name it `ComparisonPage`
2. Paste the full contents of `ComparisonPage.tsx` → **Cmd+S**

**3. Configure via Property Controls**

| Control | Default | What it changes |
|---|---|---|
| **Color Mode** | Dark | Entire template switches dark ↔ light |
| **Accent Color** | `#818CF8` | All interactive elements, highlights, CTA buttons |
| **Product Name** | Velo | Nav, hero, table header, CTA, footer — everywhere |
| **Competitor Name** | Datadog | Same behavior across all sections |

**4. Replace content**

All copy lives in data arrays at the top of `ComparisonPage.tsx`. Use the prompts in `ai-prompts/` to regenerate any section in minutes. See **[CUSTOMIZE.md](CUSTOMIZE.md)** for the full checklist.
